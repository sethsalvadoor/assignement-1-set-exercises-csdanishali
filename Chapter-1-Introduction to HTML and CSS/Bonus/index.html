HTML Tags

<p> Paragraph

The <p> tag defines a paragraph.

<p>This is a paragraph.</p>

<a> Anchor (Link)

The <a> tag creates a hyperlink.

<a href="https://example.com">Visit Example</a>

<img> Image

The <img> tag embeds an image.

<img src="image.jpg" alt="Description"/>

<ul>, <ol>, <li> Lists

Use <ul> or <ol> with <li> items.

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

<section> Section

The <section> tag defines a thematic grouping.

<section>...</section>

<header>, <footer>

Semantic containers for page header and footer.

<header>...</header>
<footer>...</footer>

<nav>

Navigation links container.

<nav>...</nav>

<code>, <pre>

Inline code and preformatted blocks.

<pre><code>Example</code></pre>

<div>, <span>

Generic block and inline containers.

<div>Block</div>
<span>Inline</span>

<h1> to <h6>

Heading levels 1 through 6.

<h1>Main Title</h1>
<h2>Subsection</h2>
