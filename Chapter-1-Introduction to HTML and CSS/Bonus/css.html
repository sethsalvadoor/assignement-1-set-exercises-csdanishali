CSS Properties

font-family
Sets the typeface.
body { font-family: 'Segoe UI', Tahoma, sans-serif; }
color
Text color.
p { color: #333; }
background-color
Element background color.
header { background-color: #005f73; }
margin, padding
Space outside/inside elements.
section { margin: 2rem; padding: 1rem; }
display
Layout behavior (block, inline, flex).
nav ul { display: flex; }
gap
Spacing between flex/grid items.
nav ul { gap: 1rem; }
border-bottom
Bottom border.
h2 { border-bottom: 2px solid #005f73; }
text-decoration
Underlines, overlines, etc.
a:hover { text-decoration: underline; }
overflow-x
Horizontal overflow behavior.
pre { overflow-x: auto; }
border-radius
Rounded corners.
code { border-radius: 3px; }
text-align
Horizontal text alignment.
footer { text-align: center; }
